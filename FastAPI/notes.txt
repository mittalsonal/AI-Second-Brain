----->>>>>  WHY VIRTUAL ENVIRONMENT ? 
-> A virtual environment creates a separate, isolated Python environment just for your project.

When you activate it:
-> python and pip now point to the project’s private environment
->Any packages you install go inside .venv/
-> Your system-wide Python stays untouched

Why this matters:
-> No version conflicts between projects
-> Clean dependency management
-> Professional project setup

-->> What is enumerate()?
enumerate() is a built-in Python function.
It gives you:
    The index
    The value
at the same time while looping.



-->>>> Difference between normal for loop and enumerate()
1️⃣ Normal for loop
for tea in teas:
    print(tea)

👉 It only return the value.
Matlab list ke andar jo item hai wahi milega.

Example:
If teas = ["Green", "Black", "Oolong"]

Output:

Green
Black
Oolong

❌ Problem:
Agar tumhe index position bhi chahiye (0,1,2), to yeh nahi dega.

2️⃣ Using enumerate()
for index, tea in enumerate(teas):
    print(index, tea)

👉 Yeh index + value dono deta hai.

Output:

0 Green
1 Black
2 Oolong
